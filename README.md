# Анализатор вегетацонных индексов (Sentinel-2)

Интерактивный веб-инструмент для расчёта 5 растительных индексов по спутниковым снимкам Sentinel-2:

- **NDVI** — нормализованный индекс растительности  
- **NDWI** — индекс воды  
- **EVI** — улучшенный NDVI  
- **SAVI** — почвенно-адаптированный индекс  
- **GNDVI** — «зелёный» NDVI  

## Установка

### Вариант 1: Docker

```bash
git clone https://github.com/yourusername/index-viewer.git
cd index-viewer
docker-compose up --build
```

[Открыть интерфейс](http://localhost:5000)

---

## Использование

1. Выбери индекс в выпадающем списке (NDVI / NDWI / EVI / SAVI / GNDVI).  
2. В правом верхнем углу карты нажми на иконку рисования.  
3. Выбери инструмент:
   - прямоугольник — выделение области
   - круг — область по радиусу
   - маркер — точка
4. Нарисуй область на карте.  
5. Приложение вычислит индекс в пределах области и отобразит:
   - цветной растр поверх карты
   - диапазон значений индекса `min … max`
   - цветовую шкалу (colorbar) снизу.
